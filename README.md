# Julebu Engine - 极简主义离线听力与句式训练系统

Julebu Engine 是一款专为高效英语学习设计的极简主义听力与句式训练平台。它结合了场景化单词学习与深度句式挑战，旨在通过高强度的打字输入与语音反馈，构建学习者的英语“肌肉记忆”。

## 核心特性
- **极简主义设计**：全黑高对比度界面，摒弃一切干扰，专注学习本身。
- **深度场景化**：从单词到完整句式的进阶式学习路径。
- **离线支持**：所有核心逻辑与基础词典均内置，性能优越。
- **设备适配**：完美适配 iPad 虚拟键盘，支持随时随地开启练习。

## 如何使用
1. 选择你感兴趣的课程主题。
2. 根据中文提示与语音反馈，在键盘上输入对应的英文内容。
3. 通过大量的重复与反馈，掌握地道的英语表达。
